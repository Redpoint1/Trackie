Dokumentácia API

V prípade potreby autorizácie potrebujete posielať HTTP požiadavky na REST API s Authorization: Token <token_id>. API token môžete získať po prihlásení z profilu.

Dôležité REST API koncové body

POST /api/v1/trackie/races/<id_pretekov>/data/

Obsah posielaných dát závisí od nastavenia pretekov. Obsah poľa data závisí od toho, ktorý typ pretekov vybral. Príklad obsahu data nájdete v detailných informáciach daného typu pretekov. Preteky sa nastavia na priebiehajúce, ak vlastník projektu pošle prvé údaje na tento koncový bod.

Príklad obsahu HTTP požiadavky na aktualizovanie informácií o prebiehajúcom preteku, ktorý:

  • Má 3 účastníkov
  • Vlastník zadal typ preteku, ktorý obsahuje len pole Tím
  • Projekciu vybral EPSG:3857 (predvolený)

DELETE /api/v1/trackie/races/<id_pretekov>/data/

Pri zavolaní tohto koncového bodu sa ukončia preteky. Netreba posielať v tele žiadny obsah.
